⚙️ How to Run the Project (Step-by-Step):-

Install Python if you haven't already.
Check version: python --version

Install Required Libraries
Open your terminal (Command Prompt/PowerShell on Windows, or Terminal on Mac/Linux) and run: pip install Flask flask-cors requests beautifulsoup4 duckduckgo-search wikipedia

Create a folder (e.g., LokKartavya).

Place server.py, site.html, and requirements.txt in that folder.

Open Terminal/Command Prompt in that folder.

Run: pip install -r requirements.txt

Run: python server.py

Open site.html in your browser.
