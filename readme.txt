>>> How to Run the Project (Step-by-Step):-

1) Install Python if you haven't already.
   Check version: python --version

2) Install Required Libraries
   Open your terminal (Command Prompt/PowerShell on Windows, or Terminal on Mac/Linux) and run:
   pip install Flask flask-cors requests beautifulsoup4 duckduckgo-search wikipedia

3) Create a folder (e.g., LokKartavya).

4) Place server.py, site.html, and requirements.txt in that folder.

5) Open Terminal/Command Prompt in that folder.

6) Run: pip install -r requirements.txt

7) Run: python server.py

8) Open site.html in your browser.
   (Do not close this terminal — the backend must keep running.)
